# 🍔 Food Delivery App

## What is this project?
This is a **Food Delivery Backend System** — similar to how Swiggy or Zomato works behind the scenes.

When you order food online:
1. You **register/login** as a user ✅
2. You **browse restaurants** near you ✅
3. You **place an order** and track it ✅

This project handles all of that! 🚀

---

## 🏗️ How it is built?
The app is split into **3 separate services** (like 3 departments in a company):

| Service | What it does |
|---|---|
| 👤 User Service | Register, Login, Manage Users |
| 🍽️ Restaurant Service | Add & Browse Restaurants |
| 📦 Order Service | Place & Track Orders |

---

## 🛠️ Technologies Used
| Technology | Purpose |
|---|---|
| Java & Spring Boot | Main programming language & framework |
| MySQL | Database to store data |
| Hibernate | Connects Java code to database |
| Spring Security | Handles login & security |
| Postman | Used to test the APIs |
| Git & GitHub | To save and share the code |

---

## 🚀 Live Demo (API Endpoints)
| Action | Method | URL |
|---|---|---|
| Register User | POST | /api/users/register |
| Add Restaurant | POST | /api/restaurants |
| Place Order | POST | /api/orders |
| Get All Orders | GET | /api/orders |
