# 🍔 Food Delivery Microservices

A scalable Food Delivery Backend built with Spring Boot Microservices.

## 🛠️ Tech Stack
- Java 21
- Spring Boot 3.2
- Microservices Architecture
- Hibernate/JPA
- Spring Security (RBAC)
- MySQL
- REST APIs
- Postman

## 📦 Services
| Service | Port | Description |
|---|---|---|
| user-service | 8081 | User Registration & Login |
| restaurant-service | 8082 | Restaurant Management |
| order-service | 8083 | Order Processing |

## 🚀 How to Run
1. Clone the repository
2. Create MySQL databases: userdb, restaurantdb, orderdb
3. Run each service separately
